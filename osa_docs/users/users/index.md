# Users Module
## Overview
The users module consists of classes and functions that handle user-related operations in the Universal EduPlatform AI module.

## Purpose
This module provides functionalities for managing users within the Universal EduPlatform AI module, including user authentication, role validation, and user data serialization. It includes classes for handling user roles, user accounts, user login and registration, as well as views for user interactions such as logging in, registering, and logging out. Additionally, it offers decorator functions for restricting access based on user roles (student, teacher, admin) and checking permissions for various user actions.
