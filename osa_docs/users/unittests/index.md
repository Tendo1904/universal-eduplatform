# Universal EduPlatform AI Module - Unit Tests

## Overview

This module contains unit tests for various components within the Universal EduPlatform AI module. It includes test cases for functionalities related to user authentication, registration, token decoding, and API views.

## Purpose

The purpose of this module is to ensure the proper functioning and reliability of the codebase by testing critical features such as login, logout, user registration, token decoding, and API views. These tests help in verifying that the implemented functionalities work as expected and handle different scenarios effectively.
