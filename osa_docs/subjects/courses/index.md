# AI Module - Courses

## Overview
The courses module contains classes and functions specifically designed for managing subjects, themes, courses, and student enrollments within the Universal EduPlatform AI module.

## Purpose
This module aims to provide functionalities for handling subjects, courses, and themes, including serialization and deserialization of related objects. It facilitates operations such as adding new subjects, listing courses, retrieving subjects and themes, and managing student enrollments effectively.
